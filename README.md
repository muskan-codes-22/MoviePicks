# 🎬 MoviePicks

MoviePicks is a React-based movie exploration web app that allows users to search, filter, and explore movies using real-time data from the TMDB API.

---

## 🚀 Features

- 🔍 Search movies using TMDB API  
- 🎞 View movie posters, titles, and ratings  
- 📊 Sort movies by rating and title  
- 🎯 Filter movies (e.g., rating ≥ 7)  
- 🌙 Dark mode toggle  
- 📈 Dashboard with basic insights (total results, average rating, top movie)  
- 📄 Movie details page with full information  

---

## 🛠 Tech Stack

- Frontend: React (Vite)  
- State Management: Redux Toolkit  
- Routing: React Router  
- API: TMDB API  
- Styling: CSS  
- Deployment: Vercel

## ⚙️ Installation & Setup

1. Clone the repository:
git clone https://github.com/muskan-codes-22/MoviePicks.git

2. Navigate to project folder:
cd MoviePicks/moviepick

3. Install dependencies:
npm install

4. Create .env file:
VITE_API_KEY=your_tmdb_api_key

5. Run the app:
npm run dev
